# scdb
学生选课管理系统

## 主要功能：
登录验证、密码找回、列表展示、信息筛选
## 相关技术：
HTML、CSS、JavaScript、jQuery、Ajax、Bootstrap、PHP、MySQL
## 项目总结：
PHP获取服务端数据进行匹配，phpmailer插件实现密码找回，tableExport插件导出表格
